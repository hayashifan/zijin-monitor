# 版本更新规划

## 核心功能

- **实时行情监控** — A股/H股双市场15秒自动刷新，红涨绿跌标准配色
- **大宗商品联动** — 国际金价、LME铜、沪铜价格与主营业务关联监控
- **公司公告抓取** — 自动获取巨潮资讯网/东方财富公告数据
- **基本面数据** — PE/PB/市值/换手率等关键指标可视化
- **量化因子分析** — 73因子LightGBM模型，IC衰减分析，回测夏普3.15
- **K线走势图** — Sina K线API，MA5/MA10均线，30日/90日切换

**安装方式：** `git clone` → `pip install -r requirements.txt` → `npm install` → `start.bat` 一键启动
