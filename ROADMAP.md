# 版本更新规划

## v1.1.0 - 数据增强

| 功能 | 说明 |
|------|------|
| 沪铜涨跌幅 | 当前change_percent为0，需补全涨跌数据 |
| 股价历史K线图 | 接入Sina K线API，展示30日/90日走势 |
| 大宗商品历史价格 | 黄金/铜价的趋势图表 |
| 基本面数据修复 | 缓存上次有效值，非交易时段不显示空 |
| 公告详情页 | 点击公告可查看PDF原文 |

## v1.2.0 - 财报分析

| 功能 | 说明 |
|------|------|
| 财报自动抓取 | 从巨潮/东方财富抓取定期报告PDF |
| LLM财报摘要 | 接入AI模型，自动提取关键财务指标 |
| 季度对比 | 营收/净利/毛利率同比环比分析 |
| 财报时间线 | 可视化展示历史财报发布节点 |
| 财报预警 | 净利大幅下滑、毛利率异常等自动提示 |

## v1.3.0 - 智能分析

| 功能 | 说明 |
|------|------|
| 股价与商品联动分析 | 铜价/金价变动对紫金股价的相关性计算 |
| 行情情绪标签 | 展示当前股价受什么因素影响 |
| 每日早报生成 | 汇总24h内关键变动，生成简报 |
| 持仓成本追踪 | 输入持仓数据，实时计算盈亏（本地存储） |

## v2.0.0 - 移动端 + 扩展

| 功能 | 说明 |
|------|------|
| PWA支持 | 添加到手机桌面，离线缓存上次数据 |
| 移动端适配 | 响应式布局优化，触控友好 |
| Docker部署 | 一键容器化部署 |
| 多股扩展 | 支持监控其他矿业股 |
| 企业微信/钉钉推送 | 重大公告/异常波动自动通知 |

## 更新节奏

| 版本 | 预计周期 | 优先级 |
|------|----------|--------|
| v1.1.0 | 1-2周 | 高 |
| v1.2.0 | 2-3周 | 高 |
| v1.3.0 | 3-4周 | 中 |
| v2.0.0 | 1-2月 | 低 |
